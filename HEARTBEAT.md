# Heartbeat Checklist — φ-Breathe Edition

## φ-Breathe Rhythm
- Base: 8600ms per breath
- Inhale (active): 8600 × φ ≈ 13.9s
- Exhale (rest): 8600ms
- Checkpoint: every 86 breaths ≈ 32.4 min

## Light Breath (every beat)
- [ ] Gateway health: `openclaw gateway status`
- [ ] Active tasks: any pending work?
- [ ] Session status: token usage reasonable?

## Deep Breath (every φ beats ≈ 14)
- [ ] Memory review: recent daily files need summarizing?
- [ ] Rolling summary: update if significant events
- [ ] Workspace cleanup: old temp files, stale artifacts

## Full Breath (every φ² beats ≈ 23)
- [ ] HEALTH-MONITOR protocol (see AGENTS.md)
- [ ] Update memory/NYAN_MEMORY_PROTOCOL.md if needed
- [ ] Review runbooks/BUILD-v2.md — any drift?

## Checks (rotate through, ~4x daily)
- [ ] Email - urgent unread?
- [ ] Calendar - events < 2h?
- [ ] Weather - relevant for Bangkok

## When to Stay Silent (HEARTBEAT_OK)
- Late night Bangkok (23:00-08:00) unless urgent
- Nothing new since last check
- Just checked < 15 min ago

## When to Reach Out
- Important email arrived
- Calendar event < 2h
- Gateway unhealthy
- Something interesting discovered
